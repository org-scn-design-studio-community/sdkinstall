Flow Layouter
This page is giving short introduction to the component with summary of the change status (green/yellow/red)

The target for this component is to achieve responsive layouting with standard design studio components (or other SDK components of course). There are some other options to make responsive layout, eg via CSS - Designing Responsive Components in Design Studio. This is fine, but it is restricted to the CSS functionality.

Link to blog: Design Studio SDK: Flow Layouter Component (for responsive layouting)

reLayout		 Re-Layout the content based on the given rules, which are:
- the size will be taken from this layouter (split is done on width)
- the space between components will be set by the property "split space"
- if "hide on overlap" is set to true, components which are outside of visible area are hidden.
- in other case, when the parent component is set, it will be resized to the new content
